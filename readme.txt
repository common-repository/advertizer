Advertizer is a Wordpress Advertizing manager, that allows you to create as many ads blocks as you need. It doesnt allow to add adsense or other context ads system. Main idea - add banners and other image oriented ads. Any image format, basic trace of clicks.

== Description ==
If you want to add banners of your partners or friends to your blog - you can use Advertizer plugin to do it easily.
You have just to create ads block in Advertizer admin. And then put it in any widget area you need.
With this plugin you can create any amount of ads blocks and put them anywhere you need.
Basic tracking of clicks and other parameters are avalible.
Additional install and configuration information is avalible <a href="http://voodoopress.net/?page_id=15">here</a> 
== Installation ==

1. Upload `advertizer` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add Ads block with parameters you need.
4. Add widgets with needed blocks to your WP theme
